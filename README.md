## ðŸŽ® How to Run

### 1\. Compile (Build)

Target a specific file and output it to the `class` folder to keep things tidy if not build yet .

```bash
# Syntax: javac -d bin src/[LabFolder]/[File].java
#example build year.java :
javac -d class src/lab1/year.java
```

### 2\. Start Game (Run)

Run the compiled class directly from the `class` folder.

```bash
# Syntax: java -cp bin [ClassName]
java -cp class/lab1 year
```
